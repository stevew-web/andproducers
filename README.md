# AndProducers (GitHub Pages + Jekyll)

This repo is a ready-to-deploy GitHub Pages site for **AndProducers**.

## Quick start (GitHub Pages)
1. Create a new GitHub repo (e.g. `andproducers`).
2. Upload/commit everything in this folder to the repo root.
3. In **Settings â†’ Pages**:
   - **Build and deployment**: Deploy from a branch
   - Branch: `main` / folder: `/ (root)`
4. Your site will appear at `https://<username>.github.io/<repo>/`

## Editing content
- Pages live in `/pages/` (Markdown).
- Posts live in `/_posts/` (optional; not required for launch).

## Notes
- This site uses a lightweight custom layout (no paid theme).
- You can add Google Analytics or Plausible in `_config.yml` and `_includes/head.html`.
